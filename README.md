## Опис
Проєкт розроблено на React + Redux Toolkit. Реалізовано сторінку аналітики з графіками та імпорт даних.

## Функціонал
1. **Візуалізація:** Використано бібліотеку Recharts для відображення кількості реєстрацій за днями.
2. **Зовнішній API:** Реалізовано імпорт користувачів з JSONPlaceholder.
3. **Адаптивність:** Графіки коректно відображаються на мобільних пристроях.

## Запуск
1. `npm install`
2. `npm run dev`
